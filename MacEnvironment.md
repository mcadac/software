# Mac Enviroment

1. Install homebrew:
  ```
  /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  ```
2. Install keepasssx
  ```
  brew cask install keepassx
  ```
3. Install Git
  ```
  brew install git
  ```
4. Install Postgresql
5. Install Intellij
